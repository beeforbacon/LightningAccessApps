# Bitcoin Lightning Network Wallets

One-page guide recommending Bitcoin Lightning wallets by type: self custody, neobanks, and exchanges.

## Host on GitHub Pages

1. **Create a new repository on GitHub**
   - Go to [github.com/new](https://github.com/new)
   - Name it (e.g. `walletrecomendations` or `lightning-wallets`)
   - Leave it empty (no README, .gitignore, or license)
   - Click **Create repository**

2. **Push this project**
   ```bash
   cd "c:\Users\NatePC\Desktop\Projects\walletrecomendations"
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
   Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your GitHub username and repo name.

3. **Turn on GitHub Pages**
   - In the repo, go to **Settings** → **Pages**
   - Under **Source**, choose **Deploy from a branch**
   - Branch: **main**, folder: **/ (root)**
   - Click **Save**

4. **View your site**
   - After a minute or two it will be at:  
     **https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/**

The main page is `index.html`, so the URL above will show the wallet recommendations page.
